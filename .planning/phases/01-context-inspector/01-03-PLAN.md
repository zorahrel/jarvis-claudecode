**File 1: `router/src/services/contextInspector/claudeMdChain.ts`**

Implement these exports (signatures EXACT â€” Plan 03 Task 2 imports them by name):

- `interface ChainEntry { path: string; bytes: number; tokens: number; isRoot: boolean }`
- `interface ChainResult { entries: ChainEntry[]; totalBytes: number; totalTokens: number; missing: string[] }`
- `async function expandClaudeMdChain(rootPath: string): Promise<ChainResult>`

Algorithm:
1. Maintain a `Set<string>` of visited absolute paths to prevent cycles.
2. Read each file's leading 4 KB only (perf + matches Claude Code preset behavior, which only follows @-imports in the directives section).
3. Match @-imports with this regex: `/^@(\S+)/gm` (multiline, anchored at line start, captures the path token).
4. For each import:
   - If starts with `~/` or equals `~`: expand to `path.join(homedir(), rest)`.
   - If `path.isAbsolute(ref)`: use as-is.
   - Else: resolve relative to `path.dirname(currentFile)`.
5. Use `fs.promises.readFile` with try/catch â€” on ENOENT, push to `missing[]` and continue.
6. Each entry: `tokens = Math.ceil(bytes / 4)`.
7. `totalBytes` = sum of entries.bytes; `totalTokens = Math.ceil(totalBytes / 4)`.
8. Return entries in DFS-from-root order: root first, then each import in the order they appeared, recursing depth-first.

Exports (signatures EXACT):

```
interface SpawnConfig {
  agent: string;
  fullAccess: boolean;
  inheritUserScope: boolean;
  tools: string[];
  workspace: string;
  mcpConfigPath: string;
  userClaudeMdPath?: string;  // defaults to path.join(homedir(), ".claude", "CLAUDE.md")
}

interface McpServerDetail { name: string; transport: "stdio" | "http" | "unknown"; toolsEstimate: number; tokens: number }
interface ChainEntryDetail { path: string; bytes: number; tokens: number; isRoot: boolean }
interface CategoryResult { category: BreakdownCategory; tokens: number; details: McpServerDetail[] | ChainEntryDetail[] | { note: string } | null }
interface BreakdownResult { categories: CategoryResult[]; totalEstimated: number; liveTotal: number }

async function calculateBreakdown(spawn: SpawnConfig, liveTotalTokens: number): Promise<BreakdownResult>
```

Implementation rules:
1. **system_preset**: hardcoded SYSTEM_PRESET_TOKENS, details = `{ note: "Claude Code preset (excludeDynamicSections=true)" }`
2. **builtin_tools**: hardcoded BUILTIN_TOOLS_TOKENS, details = `{ note: "Read/Write/Edit/Bash/Grep/Glob/Task/Agent/WebFetch/WebSearch/TodoWrite schemas" }`
3. **mcp_servers**:
   - Read `spawn.mcpConfigPath` as JSON. Server map = `parsed.mcpServers ?? parsed` (tolerate either shape).
   - If `spawn.fullAccess`: load all server names. Else: `spawn.tools.filter(t => t.startsWith("mcp:")).map(t => t.slice(4))` and intersect with available names.
   - For each loaded server, derive transport: `cfg.transport === "http" || cfg.url` => `"http"`; `cfg.command || cfg.transport === "stdio"` => `"stdio"`; else `"unknown"`.
   - Each detail: `{ name, transport, toolsEstimate: 8, tokens: MCP_AVG_TOKENS_PER_SERVER }`
   - Total tokens = `details.length * MCP_AVG_TOKENS_PER_SERVER`
4. **skills_index**: if `!inheritUserScope` -> 0 tokens, details = `{ note: "inheritUserScope=false â€” no user-scope skills" }`. Else: SKILLS_INDEX_DEFAULT_TOKENS, details = `{ note: "jarvis-marketplace 29 + user 3 + plugin marketplace; gsd:* alone ~2.5k" }`
5. **claudemd_chain**: 
   - If `inheritUserScope`: call `expandClaudeMdChain(spawn.userClaudeMdPath ?? path.join(homedir(), ".claude", "CLAUDE.md"))`, push entries.
   - Always also call `expandClaudeMdChain(path.join(spawn.workspace, "CLAUDE.md"))` and push.
   - Sum tokens across both. Details = combined entries[].
6. **subagents**: if `!inheritUserScope` -> 0. Else: SUBAGENTS_INDEX_TOKENS.
7. **hooks_memory**: if `!inheritUserScope` -> 0. Else: HOOKS_MEMORY_AVG_TOKENS.
8. **history**: `Math.max(0, liveTotalTokens - sum(other 7))`. Details = `{ note: ... }` describing how it was computed.

Return order MUST be: system_preset, builtin_tools, mcp_servers, skills_index, claudemd_chain, subagents, hooks_memory, history.

`totalEstimated` = sum of all 8 categories' tokens.
