```
interface ToolUseRecord { name: string; inputKeys: string[]; turnIndex: number }

async function readJsonlTailLines(path: string, maxBytes?: number): Promise<string[]>  // default maxBytes = 256_000
async function extractToolUseEvents(path: string, lastNTurns?: number): Promise<ToolUseRecord[]>  // default lastNTurns = 5
async function countCompactions(path: string): Promise<number>
async function sumTokens(path: string): Promise<{ input: number; output: number; cacheCreation: number; cacheRead: number; total: number }>
async function countTurns(path: string): Promise<number>  // count of role=assistant lines
```

Implementation rules:

1. **readJsonlTailLines(path, maxBytes=256_000)**:
   - Mirror the pattern in `localSessions/discovery.ts` `readJsonlTail` (lines 183-199): open file, stat size, allocate buffer of `min(size, maxBytes)`, read from `max(0, size - maxBytes)`, split by `\n`, filter empty lines.
   - On any error (ENOENT, permission), return `[]`.

2. **extractToolUseEvents(path, lastNTurns=5)**:
   - Read tail lines.
   - Walk forward, tracking `turnIndex` (incremented each time we see a `type === "assistant"` line).
   - Total turns in tail = max turnIndex seen + 1. We want only events where `turnIndex >= totalTurns - lastNTurns`.
   - For each `assistant` line, parse `message.content` array. For each block where `block.type === "tool_use"`, push `{ name: block.name, inputKeys: Object.keys(block.input ?? {}), turnIndex }`.
   - Return only entries from the last N turns.

3. **countCompactions(path)**:
   - Read full tail (or scan whole file if size < 1 MB; otherwise tail).
   - Count lines where parsed JSON has `isCompactSummary === true`.
   - Return count. On error: return 0.

4. **sumTokens(path)**:
   - Walk all assistant lines in the tail.
   - For each `message.usage` block, accumulate `input_tokens`, `output_tokens`, `cache_creation_input_tokens`, `cache_read_input_tokens`.
   - Return `{ input, output, cacheCreation, cacheRead, total: sum }`. Missing fields default to 0.

5. **countTurns(path)**:
   - Read tail. Count lines where `type === "assistant"`. Note: this is approximate (only counts what fits in tail). Return number.

All functions must be `async` and use `try/catch` with `JSON.parse` (skip malformed lines, never throw).

Exports (signatures EXACT):

```
interface DiskStats {
  totalMb: number;
  totalJsonl: number;
  filesOlderThan30d: number;
}

interface RecentSession {
  slug: string;
  sessionId: string;
  transcriptPath: string;
  cwd: string;            // best-effort reconstruction from slug (caveat: not invertible per session-discovery report)
  routeHint: string | null; // e.g. "notch" if cwd looks like ~/.claude/jarvis/agents/notch
  mtime: number;          // ms epoch
  sizeBytes: number;
  totalTokens: number;
  turnCount: number;
  compactionCount: number;
}

async function diskStats(projectsRoot: string): Promise<DiskStats>
async function recentSessions(projectsRoot: string, limit?: number): Promise<RecentSession[]>  // default limit = 10
```

**diskStats algorithm:**

1. `fs.readdir(projectsRoot, { withFileTypes: true })` to enumerate slug dirs.
2. For each slug dir, `fs.readdir(slugDir, { withFileTypes: true })`.
3. For each `.jsonl` file: `fs.stat`, accumulate `totalBytes += st.size`, `totalJsonl++`, if `Date.now() - st.mtimeMs > 30*86400_000` increment `filesOlderThan30d`.
4. Return `{ totalMb: totalBytes / 1024 / 1024, totalJsonl, filesOlderThan30d }`.
5. On any error reading projectsRoot: return zeros.

**recentSessions algorithm:**

1. Enumerate all `.jsonl` files across all slugs (collect `{ slug, name, path, st }`).
2. Sort by `st.mtimeMs` descending. Take top `limit`.
3. For each, build entry:
   - `slug` = parent dir name
   - `sessionId` = filename without `.jsonl` extension
   - `transcriptPath` = full path
   - `cwd` = best-effort reconstruction: `"/" + slug.replace(/^-/, "").replace(/-/g, "/")` (caveat noted in JSDoc)
   - `routeHint`: if `cwd` matches `/jarvis/agents/<NAME>(?:/|$)`, capture `<NAME>`. Else null.
   - `mtime` = `st.mtimeMs`
   - `sizeBytes` = `st.size`
   - `totalTokens` = `(await sumTokens(path)).total`
   - `turnCount` = `await countTurns(path)`
   - `compactionCount` = `await countCompactions(path)`
4. Use `Promise.all` for the per-file enrichment to keep it fast.
5. Return the array (already ordered by mtime desc).
